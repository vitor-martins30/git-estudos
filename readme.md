## Principais comandos

#### Cria um novo repositório para ser rastreado todas as mudanças 
```shell
git init
```

#### Exibe o status atual de todos os arquivos rastreados 
```shell
git status
```

####  
```shell
git add <nome-arquivo.extensao>
git add readme.md

git add <*.extensao>
git add *.md

git add . 
```

####  
```shell
git commit -m "sua mensagem"
```
